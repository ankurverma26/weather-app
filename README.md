# 🌤️ Weather App

A simple weather application built using **HTML**, **CSS**, and **JavaScript**. It fetches real-time weather data and displays the current weather along with a 7-day forecast.

## Features

* Search weather by city
* Current temperature and weather condition
* Feels like temperature
* Humidity
* Wind speed
* Pressure
* Visibility
* Precipitation probability
* 7-day weather forecast
* Responsive design

## Tech Stack

* HTML
* CSS
* JavaScript
* Open-Meteo API
* Open-Meteo Geocoding API

## Installation

```bash
git clone https://github.com/ankurverma26/weather-app.git

cd weather-app
```

Open `index.html` in your browser.
